# Chef Job Registration

## Description
The "Chef Job Registration" project is a guided exercise by Scrimba Front-end Program focusing on advanced responsive CSS techniques. The application serves as a registration form for aspiring chefs to apply for a position, with a design that adjusts seamlessly across various devices.

## Technologies
- HTML
- CSS
- JavaScript

## Learnings
- Adapted text case transformations using CSS for stylistic emphasis.
- Applied viewport units (vw & vh) to ensure elements scale based on different screen sizes.
- Manipulated the order of flexbox items to enhance mobile responsiveness.
- Utilized box-sizing to maintain consistent element sizing.
- Styled form elements with CSS to improve user experience and interaction.

## Setup
Clone the repository and open `index.html` on any device to fill out the registration form.
